# 1. Scala Programming Language

## What is Scala?

**Scala** is a programming language that:

- Runs on the **JVM** (Java Virtual Machine)
- Combines **Object-Oriented Programming (OOP)** and **Functional Programming (FP)**
- Is fully compatible with Java libraries
    

Created by Martin Odersky.

---

## Core Idea

Scala = **Java + Functional Programming + Better Type System**

You can:

- Write classes like Java
- Use immutable data
- Use functions as values
- Write concise, expressive code
    

---

## Very Simple Example

```scala
object Main {
  def main(args: Array[String]): Unit = {
    println("Hello, Scala!")
  }
}
```

Functional example:

```scala
val numbers = List(1, 2, 3)
val doubled = numbers.map(x => x * 2)
```

Here:

- `List` is immutable
- `map` applies a function to each element
- `x => x * 2` is a lambda (anonymous function)
    

---

## Why Use Scala?

- Strong static typing (errors caught at compile time)
- Powerful functional programming
- Great for backend and distributed systems
- Used heavily with Apache Spark (big data)

# Setup Guide (Step-by-Step)

## Option 1 — Recommended: Install via SDKMAN (Linux/macOS)

### 1. Install SDKMAN

```bash
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

SDKMAN = tool to manage JVM languages (Java, Scala, etc.)

---

### 2. Install Scala

```bash
sdk install scala
```

Check version:

```bash
scala -version
```

---

### 3. Install sbt (Build Tool)

```bash
sdk install sbt
```

Check:

```bash
sbt sbtVersion
```

`sbt` = Scala Build Tool (like Maven/Gradle for Java)

---

## Option 2 — If You Already Have Java

Check Java:

```bash
java -version
```

You need Java 8+ (preferably 17 or 21).

If not installed (Ubuntu example):

```bash
sudo apt install openjdk-17-jdk
```

Then install Scala via:

```bash
sudo apt install scala
```

(Not recommended for newest version — SDKMAN is better.)

---

# Create First Scala Project (Minimal)

## 1. Create project

```bash
mkdir my-scala-app
cd my-scala-app
sbt new scala/scala3.g8
```

Follow prompts.

---

## 2. Run project

```bash
sbt run
```

---

# Alternative: Simple Script (No Project)

Create file:

```bash
nano hello.scala
```

Content:

```scala
@main def hello(): Unit =
  println("Hello Scala 3")
```

Run:

```bash
scala hello.scala
```
